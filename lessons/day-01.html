Day 1 Number Systems & Exponents

Rational vs. Irrational Numbers

Vocabulary

Real Number
Any number that can be placed on a number line. Includes both rational and irrational numbers.
Rational Number
Any number expressible as $\frac{p}{q}$ where $p$ and $q$ are integers and $q \neq 0$. Its decimal either terminates or repeats.
Irrational Number
A real number that cannot be written as a fraction. Its decimal never terminates and never repeats.
Integer
A whole number or its negative: $\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots$
Terminating Decimal
A decimal that ends after a finite number of digits. Example: $0.75$, $3.125$.
Repeating Decimal
A decimal where one or more digits repeat forever. Example: $0.\overline{3} = 0.3333\ldots$ and $0.\overline{142857}$.

Concept

Every number you'll use in algebra is a real number. The real numbers split into exactly two groups: rational and irrational. The diagram below shows how these groups nest inside each other.

The real number system — every number is either rational or irrational, never both.

The key question is always: can this number be written as a fraction of two integers? If yes → rational. If no → irrational.

💡
Three quick tests for rational:
1. It is a fraction already (e.g. $\frac{5}{8}$).
2. Its decimal terminates (e.g. $0.25 = \frac{1}{4}$).
3. Its decimal repeats (e.g. $0.\overline{3} = \frac{1}{3}$).

Watch out for square roots. $\sqrt{9} = 3$ is rational because 9 is a perfect square. But $\sqrt{7}$ is irrational because 7 is not a perfect square — its decimal goes $2.6457513\ldots$ forever without repeating.

⚠️
$3.14$ is rational (it terminates). $\pi$ is irrational ($\pi \approx 3.14159265\ldots$ — it never ends, never repeats). They are not the same number!

Worked Examples

Example 1 — Classify a list of numbers

Classify each number as rational or irrational: $\quad\dfrac{3}{4},\quad -5,\quad \sqrt{9},\quad \sqrt{2},\quad \pi,\quad 0.1\overline{6}$

1
$\dfrac{3}{4}$ — already a fraction of two integers.  → Rational
2
$-5 = \dfrac{-5}{1}$ — integers can always be written as fractions.  → Rational
3
$\sqrt{9} = 3$ — 9 is a perfect square, the result is an integer.  → Rational
4
$\sqrt{2} = 1.41421356\ldots$ — not a perfect square, decimal never repeats.  → Irrational
5
$\pi = 3.14159265\ldots$ — famous irrational constant, decimal never repeats.  → Irrational
6
$0.1\overline{6} = 0.1666\ldots$ — decimal repeats (the 6 repeats forever).  → Rational
Rational: $\frac{3}{4},\ -5,\ \sqrt{9},\ 0.1\overline{6}$  ·  Irrational: $\sqrt{2},\ \pi$
Example 2 — Convert a fraction to decimal form

Convert $\dfrac{2}{3}$ to decimal form. Is it rational or irrational?

1
Divide numerator by denominator: $2 \div 3$.
2
Perform long division: $3$ goes into $2.000\ldots$:
$2 \div 3 = 0$ remainder $2$, bring down → $20 \div 3 = 6$ remainder $2$, bring down → $20 \div 3 = 6$ remainder $2$, …
3
The remainder always equals 2, so the digit 6 repeats forever: $\dfrac{2}{3} = 0.\overline{6}$
$\dfrac{2}{3} = 0.\overline{6} = 0.6666\ldots$  → Rational (repeating decimal)
Example 3 — Tricky case

Is $0.101001000100001\ldots$ rational or irrational?

(Each group of zeros grows by one: 1, 01, 001, 0001, …)

1
Ask: does the decimal terminate? No — it goes on forever.
2
Ask: does any block of digits repeat? No — the pattern keeps changing (1 zero, then 2 zeros, then 3 zeros, …). No fixed block repeats.
3
A number with a non-terminating, non-repeating decimal cannot be written as $\frac{p}{q}$.
Irrational — the decimal neither terminates nor repeats.

Practice Problems

Which of the following numbers is irrational?

Ask yourself: is each number under a square root sign a perfect square?
$\sqrt{16} = 4$ (perfect square → rational), $\frac{2}{7}$ is a fraction (rational), $0.25$ terminates (rational). $\sqrt{5} = 2.2360679\ldots$ — 5 is not a perfect square, so the decimal never repeats. Irrational.

Which of the following numbers is rational?

A repeating decimal can always be written as a fraction.
$\pi$, $\sqrt{3}$, and $\sqrt{11}$ are all well-known irrationals. $0.\overline{3} = \dfrac{1}{3}$ — it repeats, so it is rational.

Convert $\dfrac{1}{8}$ to decimal form. Enter the decimal.

Divide: $1 \div 8$. How many times does 8 go into 10? Into 20?
$1 \div 8 = 0.125$. The decimal terminates, confirming that $\frac{1}{8}$ is rational.

Simplify $\sqrt{36}$. What is the result?

What number times itself equals 36?
$\sqrt{36} = 6$ because $6 \times 6 = 36$. Since the result is an integer, $\sqrt{36}$ is rational.

The decimal $0.010010001\ldots$ (each group of zeros grows by one) is:

For a decimal to be rational, it must repeat the same block of digits over and over. Does this one do that?
Having a pattern is not enough. The same block must repeat. Here the zeros keep growing (1, then 2, then 3, …), so no fixed block repeats. This is a non-repeating, non-terminating decimal → irrational.

Which set contains only rational numbers?

Check every number in each set. One irrational number disqualifies the whole set.
Set A has $\sqrt{2}$ (irrational). Set B has $\pi$ (irrational). Set D has three irrationals. Set C: $-3 = \frac{-3}{1}$ ✓, $0 = \frac{0}{1}$ ✓, $\frac{4}{5}$ ✓ — all rational.

Write $0.4$ as a fraction in lowest terms. (Enter as a fraction, e.g. 3/4)

$0.4 = \dfrac{4}{10}$. Can you simplify that fraction?
$0.4 = \dfrac{4}{10} = \dfrac{2}{5}$. Divide numerator and denominator by their GCF, which is 2.

Between any two rational numbers, there is always:

Try finding a number between $\frac{1}{2}$ and $\frac{3}{4}$. Can you always find another one after that?
Between any two rationals $a$ and $b$, the average $\dfrac{a+b}{2}$ is also rational. You can repeat this process infinitely — so there are infinitely many rationals between any two rationals. The rationals are dense on the number line.

Simplify $-\sqrt{49}$. What is the result? (Negative square roots are still real numbers — the negative sign is outside the radical.)

First simplify the radical: $\sqrt{49} = ?$ Then apply the negative sign.
$\sqrt{49} = 7$ because $7 \times 7 = 49$. Then $-\sqrt{49} = -7$. The result is the integer $-7$, which is rational. The negative sign outside the radical does not make the number irrational.

Which of the following expressions equals a rational number?

Evaluate each expression. Does any one simplify to a number with no radical or $\pi$?
$\sqrt{3} + \sqrt{3} = 2\sqrt{3}$ — still irrational.
$\sqrt{5} \times \sqrt{5} = 5$ — the radicals cancel, leaving the integer 5. Rational.
$\sqrt{2} + 1 \approx 2.414\ldots$ — irrational plus rational stays irrational.
$\pi - 1 \approx 2.14159\ldots$ — irrational minus rational stays irrational.
Key insight: two irrational numbers can multiply to give a rational result.
