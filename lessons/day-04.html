Day 4 Number Systems & Exponents

Exponents & Radicals Review

Quick-Reference Summary

Exponent Rules
$a^m \cdot a^n = a^{m+n}$
$\dfrac{a^m}{a^n} = a^{m-n}$
$(a^m)^n = a^{mn}$
$(ab)^n = a^n b^n$
$a^0 = 1 \quad (a \neq 0)$
$a^{-n} = \dfrac{1}{a^n}$
Radical Rules
$\sqrt[n]{a} \cdot \sqrt[n]{b} = \sqrt[n]{ab}$
$\dfrac{\sqrt[n]{a}}{\sqrt[n]{b}} = \sqrt[n]{\dfrac{a}{b}}$
$a^{1/n} = \sqrt[n]{a}$
$a^{m/n} = \left(\sqrt[n]{a}\right)^m$
Like radicals: $p\sqrt{a} + q\sqrt{a} = (p+q)\sqrt{a}$
🔑
Strategy for complex simplifications: (1) Expand any powers of products/quotients first. (2) Combine terms with the same base using product/quotient rules. (3) Convert between radical and rational-exponent form if it helps. (4) Check that the final answer has no negative exponents and all radicals are simplified.

Adding & Subtracting Radicals

You can only add or subtract radicals that have the same index and the same radicand (called like radicals). Think of $\sqrt{3}$ as a "unit," just like $x$ in algebra. $2\sqrt{3} + 5\sqrt{3} = 7\sqrt{3}$ for the same reason $2x + 5x = 7x$.

Always simplify first — radicals that look different may become like radicals after simplification. For example, $\sqrt{12} + \sqrt{27}$ looks like unlike radicals, but:

$$\sqrt{12} + \sqrt{27} = 2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$$
⚠️
$\sqrt{12} + \sqrt{27} \neq \sqrt{39}$. You cannot add the radicands — only combine like radicals after simplifying.

Worked Examples

Example 1 — Add Radicals After Simplifying

Simplify $\sqrt{20} + \sqrt{45}$.

1
Simplify $\sqrt{20}$: largest perfect-square factor is 4. $\sqrt{20} = \sqrt{4 \cdot 5} = 2\sqrt{5}$
2
Simplify $\sqrt{45}$: largest perfect-square factor is 9. $\sqrt{45} = \sqrt{9 \cdot 5} = 3\sqrt{5}$
3
Now they are like radicals: $2\sqrt{5} + 3\sqrt{5} = 5\sqrt{5}$
$\sqrt{20} + \sqrt{45} = 5\sqrt{5}$
Example 2 — Multi-Rule Exponent Simplification

Simplify $\dfrac{(x^3)^2 \cdot x^{-1}}{x^5}$

1
Power rule in numerator: $(x^3)^2 = x^6$
2
Product rule: $x^6 \cdot x^{-1} = x^{6+(-1)} = x^5$
3
Quotient rule: $\dfrac{x^5}{x^5} = x^{5-5} = x^0 = 1$
$\dfrac{(x^3)^2 \cdot x^{-1}}{x^5} = 1$
Example 3 — Rational Exponent with Fraction Base

Evaluate $\left(\dfrac{1}{2}\right)^{-3}$.

1
Negative exponent means take the reciprocal of the base: $\left(\dfrac{1}{2}\right)^{-3} = \left(\dfrac{2}{1}\right)^3 = 2^3$
2
Evaluate: $2^3 = 8$
$\left(\dfrac{1}{2}\right)^{-3} = 8$

Practice Problems

Simplify $2\sqrt{5} + 3\sqrt{5}$.

Like radicals: treat $\sqrt{5}$ as a unit, like a variable. $2x + 3x = ?$
$2\sqrt{5} + 3\sqrt{5} = (2+3)\sqrt{5} = 5\sqrt{5}$. The radicand stays the same — you add coefficients only.

Simplify $\dfrac{x^{-2} \cdot x^5}{x^0}$.

Simplify the denominator first ($x^0 = ?$), then combine the numerator with the product rule.
$x^0 = 1$, so denominator is 1. Numerator: $x^{-2} \cdot x^5 = x^{-2+5} = x^3$. Result: $x^3$.

Simplify $\sqrt{x^{10}}$ (assume $x \ge 0$). Write your answer using an exponent.

$\sqrt{x^{10}} = (x^{10})^{1/2}$. Use the power rule: multiply the exponents.
$\sqrt{x^{10}} = (x^{10})^{1/2} = x^{10 \cdot \frac{1}{2}} = x^5$.

Simplify $\dfrac{1}{\sqrt{4}}$.

Simplify the denominator first: $\sqrt{4} = ?$
$\sqrt{4} = 2$, so $\dfrac{1}{\sqrt{4}} = \dfrac{1}{2}$.

Simplify $\dfrac{x^2}{x^{-3}}$.

Quotient rule: subtract the exponents. Be careful — you're subtracting a negative number.
$\dfrac{x^2}{x^{-3}} = x^{2-(-3)} = x^{2+3} = x^5$. Subtracting a negative adds.

Write $4.7 \times 10^{-3}$ in standard (decimal) notation.

A negative exponent on 10 means move the decimal to the left. How many places?
$10^{-3} = 0.001$. Move the decimal 3 places left: $4.7 \to 0.0047$.

Simplify $\sqrt{12} + \sqrt{27}$.

Simplify each radical first. What is the largest perfect-square factor of 12? Of 27?
$\sqrt{12} = \sqrt{4 \cdot 3} = 2\sqrt{3}$  ·  $\sqrt{27} = \sqrt{9 \cdot 3} = 3\sqrt{3}$
Now like radicals: $2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$.

Evaluate $\left(\dfrac{1}{2}\right)^{-3}$.

A negative exponent flips the fraction. What is the reciprocal of $\dfrac{1}{2}$?
$\left(\dfrac{1}{2}\right)^{-3} = \left(\dfrac{2}{1}\right)^3 = 2^3 = 8$. The negative exponent reciprocates the fraction; the result is positive.

Evaluate $16^{3/4}$.

$a^{m/n} = \left(\sqrt[n]{a}\right)^m$. Take the 4th root of 16 first, then cube it.
$16^{3/4} = \left(\sqrt[4]{16}\right)^3 = 2^3 = 8$. Check: $2^4 = 16$ ✓, then $2^3 = 8$.

Simplify $\dfrac{\sqrt{50x^4}}{x}$ (assume $x > 0$).

Simplify the numerator first: factor out perfect squares from both the number and the variable part.
$\sqrt{50x^4} = \sqrt{25 \cdot 2 \cdot x^4} = \sqrt{25} \cdot \sqrt{x^4} \cdot \sqrt{2} = 5x^2\sqrt{2}$
Then divide: $\dfrac{5x^2\sqrt{2}}{x} = 5x\sqrt{2}$.
