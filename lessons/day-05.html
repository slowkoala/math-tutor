Day 5 Linear Equations & Graphing

Solving Linear Equations

Vocabulary

Linear Equation
An equation where the variable has exponent 1. Its graph is a straight line. General form: $ax + b = c$.
Solution
A value of the variable that makes the equation true. To check, substitute it back in and verify both sides are equal.
Inverse Operations
Operations that undo each other: addition ↔ subtraction, multiplication ↔ division. Used to isolate the variable.
Properties of Equality
Whatever operation you apply to one side of an equation, you must apply to the other side to keep it balanced.
Distributive Property
$a(b + c) = ab + ac$. Expand parentheses before combining like terms.
Like Terms
Terms with the same variable and exponent. They can be combined: $3x + 5x = 8x$. Constants are also like terms.

Concept

Solving a linear equation means isolating the variable — getting it alone on one side. The key principle: an equation is a balance. Any operation applied to one side must be applied to the other.

Each step undoes one operation until $x$ stands alone.

General strategy (works every time):

  1. Distribute to clear any parentheses.
  2. Combine like terms on each side.
  3. Move all variable terms to one side (add/subtract).
  4. Move all constants to the other side (add/subtract).
  5. Divide both sides by the coefficient of the variable.
  6. Check: substitute your answer back into the original equation.
✅
Always check your answer. Substitute it back in. If both sides equal the same number, you're done. Example: $x = 3$ in $2x + 3 = 9$: $2(3) + 3 = 9$ ✓
⚠️
When you move a term across the equals sign by adding or subtracting, its sign changes. $2x + 5 = 13$ → subtract 5: $2x = 8$. The $+5$ becomes $-5$ on the other side.

Worked Examples

Example 1 — Two-Step Equation

Solve: $3x - 7 = 14$

1
Add 7 to both sides: $3x - 7 + 7 = 14 + 7 \;\Rightarrow\; 3x = 21$
2
Divide both sides by 3: $x = \dfrac{21}{3} = 7$
3
Check: $3(7) - 7 = 21 - 7 = 14$ ✓
$x = 7$
Example 2 — Variables on Both Sides

Solve: $5x + 2 = 2x + 14$

1
Move variable terms left — subtract $2x$ from both sides: $5x - 2x + 2 = 14 \;\Rightarrow\; 3x + 2 = 14$
2
Subtract 2 from both sides: $3x = 12$
3
Divide by 3: $x = 4$
4
Check: $5(4)+2 = 22$ and $2(4)+14 = 22$ ✓
$x = 4$
Example 3 — Distributive Property

Solve: $3(x - 4) = 2x + 1$

1
Distribute: $3x - 12 = 2x + 1$
2
Subtract $2x$: $x - 12 = 1$
3
Add 12: $x = 13$
4
Check: $3(13-4) = 3(9) = 27$ and $2(13)+1 = 27$ ✓
$x = 13$

Practice Problems

Solve: $x + 7 = 15$

Subtract 7 from both sides.
$x + 7 - 7 = 15 - 7 \;\Rightarrow\; x = 8$. Check: $8 + 7 = 15$ ✓

Solve: $4x = 28$

Divide both sides by 4.
$\dfrac{4x}{4} = \dfrac{28}{4} \;\Rightarrow\; x = 7$. Check: $4(7) = 28$ ✓

Solve: $2x - 5 = 11$

Add 5 to both sides first, then divide.
$2x = 11 + 5 = 16 \;\Rightarrow\; x = 8$. Check: $2(8) - 5 = 11$ ✓

Solve: $\dfrac{x}{3} + 2 = 7$

Subtract 2 from both sides first: $\dfrac{x}{3} = 5$. Then multiply both sides by 3.
Subtract 2: $\dfrac{x}{3} = 5$. Multiply by 3: $x = 15$. Check: $\frac{15}{3} + 2 = 7$ ✓

Solve: $5x + 3 = 3x + 11$

Subtract $3x$ from both sides to collect the variable on one side.
$5x - 3x + 3 = 11 \;\Rightarrow\; 2x + 3 = 11 \;\Rightarrow\; 2x = 8 \;\Rightarrow\; x = 4$. Check: $23 = 23$ ✓

Solve: $4(x + 3) = 28$

Distribute first: $4 \cdot x + 4 \cdot 3 = 28$.
$4x + 12 = 28 \;\Rightarrow\; 4x = 16 \;\Rightarrow\; x = 4$. Check: $4(4+3) = 4(7) = 28$ ✓

Which value is a solution to $2x - 1 = 5$?

Try substituting each value into $2x - 1$ and see which gives 5.
$x=2$: $2(2)-1=3 \neq 5$.   $x=3$: $2(3)-1=5$ ✓.   $x=4$: $2(4)-1=7 \neq 5$.

Solve: $\dfrac{2x + 4}{3} = 6$

Multiply both sides by 3 first to clear the fraction.
Multiply by 3: $2x + 4 = 18 \;\Rightarrow\; 2x = 14 \;\Rightarrow\; x = 7$. Check: $\frac{18}{3} = 6$ ✓

Solve: $7 - 3x = 1$

Subtract 7 from both sides first: $-3x = -6$. Then divide by $-3$.
$-3x = 1 - 7 = -6 \;\Rightarrow\; x = \dfrac{-6}{-3} = 2$. Check: $7-3(2)=1$ ✓

Solve: $3(2x - 1) = 2(x + 5)$

Distribute both sides first, then collect variable terms on the left and constants on the right.
$6x - 3 = 2x + 10 \;\Rightarrow\; 4x = 13 \;\Rightarrow\; x = \dfrac{13}{4}$
Check: $3\!\left(2\cdot\frac{13}{4}-1\right) = 3\cdot\frac{22}{4} = \frac{33}{2}$ and $2\!\left(\frac{13}{4}+5\right) = 2\cdot\frac{33}{4} = \frac{33}{2}$ ✓
