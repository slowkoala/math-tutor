Day 6 Linear Equations & Graphing

Literal Equations & Special Cases

Vocabulary

Literal Equation
An equation containing two or more variables. You solve it by isolating one variable in terms of the others.
Formula
A literal equation describing a real-world relationship. Examples: $A = lw$, $d = rt$, $P = 2l + 2w$.
No Solution
An equation that is false for every value of the variable. Solving produces a contradiction such as $3 = 7$.
Identity
An equation that is true for every value of the variable. Solving produces a tautology such as $4 = 4$. The solution set is all real numbers.
Contradiction
A statement that is always false, such as $0 = 5$. Signals that an equation has no solution.
Coefficient
The numerical factor multiplying a variable. In $5x$, the coefficient is 5. In $-x$, it is $-1$.

Concept

Literal equations are solved using the exact same steps as regular equations — the only difference is that the answer is an expression containing other variables rather than a number. Treat every variable except the one you're solving for as if it were a constant.

🔧
Example — solve $A = lw$ for $l$:
Divide both sides by $w$: $\dfrac{A}{w} = l$, so $l = \dfrac{A}{w}$.
Same step you'd take to isolate a number — just divide by $w$ instead of a digit.

Special cases occur when all the variable terms cancel during solving. What's left is a statement involving only constants — and that statement is either always true or always false.

Three possible outcomes for a linear equation — previewing what you'll see geometrically in systems of equations.

🔀
How to identify special cases while solving:
If the variable terms cancel and you get a false constant statement (e.g. $3 = 7$) → no solution.
If the variable terms cancel and you get a true constant statement (e.g. $4 = 4$) → infinite solutions.
⚠️
When solving a literal equation, you can't get a purely numeric answer — that's fine. The answer is an expression. Example: solving $A = \frac{1}{2}bh$ for $h$ gives $h = \frac{2A}{b}$, not a number.

Worked Examples

Example 1 — Literal Equation (Formula)

Solve $P = 2l + 2w$ for $l$. (Perimeter of a rectangle.)

1
Subtract $2w$ from both sides: $P - 2w = 2l$
2
Divide both sides by 2: $l = \dfrac{P - 2w}{2}$
$l = \dfrac{P - 2w}{2}$
Example 2 — No Solution

Solve: $4x + 2 = 4x - 5$

1
Subtract $4x$ from both sides: $2 = -5$
2
$2 = -5$ is a false statement — no value of $x$ can make this true.
No solution (contradiction: $2 \neq -5$)
Example 3 — Infinite Solutions

Solve: $3(x + 4) = 3x + 12$

1
Distribute: $3x + 12 = 3x + 12$
2
Subtract $3x$: $12 = 12$
3
$12 = 12$ is always true — every real number is a solution.
Infinite solutions — all real numbers (identity)

Practice Problems

Solve $d = rt$ for $t$.

$t$ is being multiplied by $r$. What operation undoes multiplication?
Divide both sides by $r$: $\dfrac{d}{r} = \dfrac{rt}{r} \Rightarrow t = \dfrac{d}{r}$.

Solve $A = \dfrac{1}{2}bh$ for $h$. (Area of a triangle.)

Multiply both sides by 2 first to clear the fraction, then divide by $b$.
Multiply by 2: $2A = bh$. Divide by $b$: $h = \dfrac{2A}{b}$.

Solve $y = mx + b$ for $x$.

Subtract $b$ from both sides first, then divide by $m$.
$y - b = mx \;\Rightarrow\; x = \dfrac{y-b}{m}$.

How many solutions does $5x + 3 = 5x + 3$ have?

Try subtracting $5x$ from both sides. What's left?
Subtract $5x$: $3 = 3$. This is always true — every value of $x$ satisfies the original equation. Identity → infinite solutions.

How many solutions does $2x + 7 = 2x - 1$ have?

Subtract $2x$ from both sides. Is the resulting constant statement true or false?
Subtract $2x$: $7 = -1$. This is always false — no value of $x$ makes the equation true. No solution.

Solve $C = 2\pi r$ for $r$. (Circumference of a circle.)

$r$ is being multiplied by $2\pi$. Divide both sides by $2\pi$.
Divide both sides by $2\pi$: $r = \dfrac{C}{2\pi}$.

Solve $P = a + b + c$ for $b$. (Perimeter of a triangle.)

Subtract $a$ and $c$ from both sides to isolate $b$.
$P - a - c = b$, so $b = P - a - c$.

Solve $V = lwh$ for $w$. (Volume of a rectangular prism.)

$w$ is being multiplied by both $l$ and $h$. Divide both sides by $lh$.
Divide both sides by $lh$: $w = \dfrac{V}{lh}$.

How many solutions does $3(2x + 1) = 6x + 4$ have?

Distribute the left side first, then subtract $6x$.
Distribute: $6x + 3 = 6x + 4$. Subtract $6x$: $3 = 4$. False statement → no solution.

Solve $F = \dfrac{9}{5}C + 32$ for $C$. (Convert Fahrenheit to Celsius.)

Subtract 32 from both sides first, then multiply both sides by $\dfrac{5}{9}$ (the reciprocal of $\dfrac{9}{5}$).
$F - 32 = \dfrac{9}{5}C \;\Rightarrow\; C = \dfrac{5}{9}(F - 32) = \dfrac{5(F-32)}{9}$.
Option A uses the wrong fraction ($\frac{9}{5}$ instead of $\frac{5}{9}$). Option B distributes incorrectly.
