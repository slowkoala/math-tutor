Day 11 Inequalities

Two-Variable & Compound Inequalities

Vocabulary

Two-Variable Inequality
An inequality with two variables, such as $y < 2x + 1$. Its solution is an entire region of the coordinate plane, not a single value or line.
Half-Plane
The region on one side of a boundary line in the coordinate plane. The solution to a two-variable linear inequality is always a half-plane.
Boundary Line
The line $y = mx + b$ dividing the plane into two halves. Draw it dashed for strict inequalities ($<$, $>$) and solid for non-strict ($\leq$, $\geq$).
Test Point
A convenient point (usually the origin) substituted into the inequality to determine which half-plane to shade.
Conjunction (AND)
A compound inequality requiring both conditions to hold simultaneously. The solution is the intersection of the two solution sets. Example: $1 \leq x \leq 5$.
Disjunction (OR)
A compound inequality requiring at least one condition to hold. The solution is the union of the two solution sets. Example: $x < -2$ or $x > 3$.

Concept

Graphing a two-variable inequality follows three steps: (1) graph the boundary line (dashed or solid), (2) pick a test point not on the line, (3) if the test point satisfies the inequality, shade its side; otherwise shade the other side.

Dashed line (strict $<$); shaded below. Test point $(0, 0)$: $0 < 0 + 1$ ✓ — shade the side containing $(0, 0)$.

Compound inequalities combine two conditions:

AND (Conjunction)

Both conditions must be true.
Solution = intersection.

$-1 \leq x < 3$
means $x \geq -1$ AND $x < 3$

OR (Disjunction)

At least one condition must be true.
Solution = union.

$x < -2$ or $x > 3$
includes both separate rays

🔍
Which side to shade? Always test $(0, 0)$ first (unless the boundary passes through the origin, then use $(1, 0)$). If $(0, 0)$ satisfies the inequality, shade the side containing it. If not, shade the other side.

Worked Examples

Example 1 — Graph a Two-Variable Inequality

Graph $y \geq -x + 2$.

1
Graph the boundary line $y = -x + 2$. Use a solid line because the inequality is $\geq$ (includes equal).
2
Test $(0, 0)$: $0 \geq -0 + 2 = 2$? $0 \geq 2$ is false. So shade the other side (above the line).
3
The solution is the solid line and the region above it (where $y$ values are larger than $-x + 2$).
Solid line $y = -x + 2$; shade above
Example 2 — Solve a Compound AND Inequality

Solve: $3 < 2x + 1 \leq 11$

1
Subtract 1 from all three parts: $2 < 2x \leq 10$
2
Divide all parts by 2: $1 < x \leq 5$
3
Solution: $x$ is greater than 1 (open) and at most 5 (closed).
$1 < x \leq 5$
Example 3 — Solve a Compound OR Inequality

Solve: $2x - 1 > 5$ or $3x + 2 < -4$

1
Solve first: $2x > 6 \Rightarrow x > 3$.
2
Solve second: $3x < -6 \Rightarrow x < -2$.
3
OR combines with union: $x < -2$ or $x > 3$. This is two separate rays on the number line with a gap in the middle.
$x < -2$ or $x > 3$

Practice Problems

Is the point $(2, 3)$ a solution to $y < 2x + 1$?

Substitute: $y < 2x + 1 \Rightarrow 3 < 2(2) + 1$. Evaluate the right side.
$2(2)+1 = 5$. Is $3 < 5$? Yes ✓. The point $(2,3)$ is in the shaded region.

When graphing $y > 3x - 2$, how should the boundary line be drawn?

Is the boundary included in the solution? Check whether the symbol is strict ($>$) or non-strict ($\geq$).
Strict $>$ means points on the line are NOT solutions → dashed boundary line.

Which value is in the solution of $-3 \leq x < 5$?

$x$ must satisfy both $x \geq -3$ AND $x < 5$. Check each option.
$-3 \leq 0 < 5$ ✓. Note: $x=5$ fails because the right bound is strict ($<5$).

The compound inequality "$x > 2$ OR $x < -1$" — which value is not a solution?

OR: at least one condition must hold. Find the value that is neither $>2$ nor $<-1$.
$x=0$: $0>2$? No. $0<-1$? No. Neither condition holds → NOT a solution.

The compound inequality "$x \geq 1$ AND $x \leq 4$" is equivalent to:

AND means intersection: both conditions must hold simultaneously.
The AND of $x \geq 1$ and $x \leq 4$ is the segment between them: $1 \leq x \leq 4$.

Is $(0, 4)$ in the solution of $y \geq -x + 3$?

Substitute: $4 \geq -(0) + 3$. Simplify the right side.
$4 \geq 3$ ✓ (strictly greater, not just on the boundary). Yes, $(0,4)$ is in the solution.

When graphing $y < 2x - 1$, which region do you shade?

Test $(0, 0)$: $0 < 2(0)-1 = -1$? Is $0 < -1$? No → shade the other side from $(0,0)$.
$(0,0)$ fails: $0 \not< -1$. Shade the side that does NOT contain the origin — which is below the line.

Solve: $x + 2 \leq 7$ AND $x - 1 \geq 0$

Solve each part: $x \leq 5$ and $x \geq 1$. AND means both must hold.
$x \leq 5$ AND $x \geq 1$ → intersection: $1 \leq x \leq 5$.

Solve $3 < 2x + 1 \leq 9$. The solution is $? < x \leq 4$. What is the lower bound?

Subtract 1 from all parts: $2 < 2x \leq 8$. Then divide by 2.
$3-1 < 2x \leq 9-1 \Rightarrow 2 < 2x \leq 8 \Rightarrow 1 < x \leq 4$. Lower bound is $1$ (open/strict).

"$x \geq 3$ OR $x < 0$" — which value is NOT in the solution set?

OR: find values that fail both $x \geq 3$ AND $x < 0$.
$x=1$: $1 \geq 3$? No. $1 < 0$? No. Neither holds → $x=1$ is not a solution. The gap between the two rays ($0 \leq x < 3$) is excluded.
