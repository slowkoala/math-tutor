Day 12 Systems of Equations

Systems of Equations: Substitution

Vocabulary

System of Equations
Two or more equations with the same variables considered simultaneously. The goal is to find values that satisfy all equations at once.
Solution to a System
An ordered pair $(x, y)$ that makes every equation in the system true. Graphically, it is the intersection point of the lines.
Substitution Method
Solve one equation for one variable, then replace that variable in the other equation. Reduces two equations in two unknowns to one equation in one unknown.
Consistent System
A system with at least one solution — the lines intersect (one solution) or coincide (infinite solutions).
Inconsistent System
A system with no solution — the lines are parallel and never intersect. Solving produces a false constant (e.g., $0 = 5$).
Dependent System
A system with infinite solutions — both equations represent the same line. Solving produces a true constant (e.g., $0 = 0$).

Concept

A system of equations is solved when you find the point of intersection of the two lines. The substitution method works algebraically: express one variable in terms of the other, then substitute.

The solution $(2, 4)$ is the intersection of $y = 2x$ and $y = x + 2$. Both lines pass through this single point.

🔢
Substitution steps:
1. Solve one equation for one variable.
2. Substitute the expression into the other equation.
3. Solve the resulting single-variable equation.
4. Back-substitute to find the second variable.
5. Check both values in both original equations.
⚠️
After solving for $x$, don't forget to find $y$. The solution is an ordered pair $(x, y)$. Always substitute back into one of the original equations to get $y$.

Worked Examples

Example 1 — Direct Substitution

Solve: $y = 2x$ and $3x + y = 15$

1
The first equation already isolates $y$: $y = 2x$.
2
Substitute into the second: $3x + (2x) = 15 \Rightarrow 5x = 15 \Rightarrow x = 3$.
3
Back-substitute: $y = 2(3) = 6$.
4
Check: $3(3) + 6 = 15$ ✓
$(3, 6)$
Example 2 — Solve First, Then Substitute

Solve: $x + 2y = 8$ and $3x - y = 3$

1
Solve the first for $x$: $x = 8 - 2y$.
2
Substitute into second: $3(8-2y) - y = 3 \Rightarrow 24 - 6y - y = 3 \Rightarrow -7y = -21 \Rightarrow y = 3$.
3
Back-substitute: $x = 8 - 2(3) = 2$.
4
Check: $2 + 2(3) = 8$ ✓ and $3(2) - 3 = 3$ ✓
$(2, 3)$
Example 3 — No Solution (Parallel Lines)

Solve: $y = 3x + 1$ and $y = 3x - 4$

1
Both equations have $y$ isolated. Set them equal: $3x + 1 = 3x - 4$.
2
Subtract $3x$: $1 = -4$. This is a contradiction — never true.
3
Both lines have slope 3 but different intercepts → parallel → no intersection.
No solution (inconsistent system)

Practice Problems

Solve by substitution: $y = 2x$ and $3x + y = 15$.

Replace $y$ with $2x$ in the second equation: $3x + 2x = 15$.
$5x = 15 \Rightarrow x=3$, then $y=2(3)=6$. Solution: $(3,6)$.

Solve: $y = x + 3$ and $2x + y = 9$.

Substitute $y = x+3$ into $2x+y=9$: $2x+(x+3)=9$.
$3x+3=9 \Rightarrow x=2$, then $y=2+3=5$. Solution: $(2,5)$.

Solve: $y = 3$ and $4x + 2y = 18$. What is $x$?

Substitute $y=3$ into the second equation: $4x + 6 = 18$.
$4x + 6 = 18 \Rightarrow 4x = 12 \Rightarrow x = 3$. Solution: $(3, 3)$.

Solve: $x = 2y$ and $x + 3y = 10$.

Replace $x$ with $2y$ in the second equation: $2y + 3y = 10$.
$5y=10 \Rightarrow y=2$, then $x=2(2)=4$. Solution: $(4,2)$.

Is $(1, -2)$ a solution to the system $y = -2x$ and $x + y = -1$?

Check both: (1) $-2 = -2(1)$? (2) $1 + (-2) = -1$?
$-2 = -2$ ✓ and $-1 = -1$ ✓. Both equations are satisfied → $(1,-2)$ is the solution.

Solve: $y = -x + 4$ and $y = 2x - 2$.

Both expressions equal $y$. Set them equal to each other: $-x+4=2x-2$.
$6=3x \Rightarrow x=2$, then $y=-2+4=2$. Solution: $(2,2)$.

Solve: $x + y = 7$ and $y = x + 1$. What is $y$?

Substitute $y=x+1$ into $x+y=7$: $x+(x+1)=7$. Solve for $x$, then find $y$.
$2x+1=7 \Rightarrow x=3$, then $y=3+1=4$.

Solve: $2x + y = 10$ and $y = 4$.

Substitute $y=4$ into $2x+y=10$: $2x+4=10$.
$2x=6 \Rightarrow x=3$. Solution: $(3,4)$.

How many solutions does the system $y = 2x + 1$ and $y = 2x - 3$ have?

Both lines have slope 2. What does the same slope mean geometrically?
Both lines have slope 2 but different $y$-intercepts → parallel. Substituting: $2x+1=2x-3 \Rightarrow 1=-3$ (contradiction). No solution.

Solve: $3x - y = 5$ and $y = x - 1$.

Substitute $y=x-1$ into $3x-y=5$: $3x-(x-1)=5$.
$3x-x+1=5 \Rightarrow 2x=4 \Rightarrow x=2$, then $y=2-1=1$. Solution: $(2,1)$.
