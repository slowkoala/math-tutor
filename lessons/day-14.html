Day 14 Systems of Equations

Systems of Inequalities

Vocabulary

System of Inequalities
Two or more inequalities considered together; a solution must satisfy ALL inequalities simultaneously.
Feasible Region
The set of all points that satisfy every inequality in the system — the overlapping shaded area.
Corner Point (Vertex)
A point where two boundary lines intersect at the edge of the feasible region.
Boundary Line
The line that forms the edge of a half-plane. Solid if ≤ or ≥; dashed if < or >.
Half-Plane
All points on one side of a boundary line; a single inequality shades one half-plane.
No Solution
When the shaded regions of the inequalities do not overlap, the system has no solution.

Concept

To solve a system of inequalities, graph each inequality separately, then identify the overlapping region.

💡
Steps:
  1. Graph the boundary line for each inequality (solid or dashed).
  2. Shade the correct half-plane for each inequality.
  3. The feasible region is where ALL shaded areas overlap.
  4. Test a corner point to verify it satisfies all inequalities.

The purple region satisfies both $y \le x+1$ and $y \ge -x+3$.

💡
Test the corner point (1, 2):
$y \le x+1$: $2 \le 1+1 = 2$ ✓
$y \ge -x+3$: $2 \ge -1+3 = 2$ ✓

Worked Examples

Example 1 — Graph a system and test a point

Graph the system $\begin{cases}y < 2x \\ y \ge -1\end{cases}$ and identify if $(3,1)$ is a solution.

1
Graph $y = 2x$: dashed (strict <), passes through origin with slope 2. Shade below.
2
Graph $y = -1$: solid (≥), horizontal line. Shade above.
3
Feasible region: below the dashed line AND above (or on) $y=-1$.
4
Test $(3,1)$:
$1 < 2(3) = 6$ ✓
$1 \ge -1$ ✓
Yes, $(3,1)$ is in the feasible region.
$(3,1)$ satisfies both inequalities — it is in the feasible region.
Example 2 — Find the corner point of a system

Find the corner point of: $\begin{cases}y \le -x+5 \\ y \le x+1\end{cases}$

1
Set equal to find intersection: $-x+5 = x+1$
2
$4 = 2x \Rightarrow x = 2$
3
$y = 2+1 = 3$. Corner point: $(2, 3)$.
4
Verify in both:
$3 \le -2+5=3$ ✓ and $3 \le 2+1=3$ ✓
Corner point: $(2, 3)$.
Example 3 — Identify a system with no solution

Does the system $\begin{cases}y > x+2 \\ y < x-1\end{cases}$ have a solution?

1
Both boundary lines have slope 1, so they are parallel.
2
$y > x+2$ shades above the upper line; $y < x-1$ shades below the lower line.
3
The two shaded regions are on opposite sides of parallel lines — they never overlap.
No solution — the shaded regions do not overlap.

Practice Problems

Is $(0,0)$ a solution of $\begin{cases}y < x+1 \\ y \ge -1\end{cases}$?

Substitute $x=0, y=0$ into each inequality and check both conditions.
Check: $0 < 0+1 = 1$ ✓ and $0 \ge -1$ ✓. Both inequalities are satisfied, so $(0,0)$ is in the feasible region.

Is $(2,3)$ a solution of $\begin{cases}y \le 2x \\ y > x+1\end{cases}$?

Test each inequality separately: $3 \le 2(2)$? Then $3 > 2+1$?
$3 \le 4$ ✓ (passes first). But $3 > 2+1 = 3$? No — $3 \not> 3$ (strict inequality). So it fails the second inequality only.

For the inequality $y \le x+3$, should the boundary line be solid or dashed?

The ≤ symbol means "less than or equal to" — the boundary itself is included in the solution.
Use a solid line when the inequality is ≤ or ≥ (boundary included). Use a dashed line for strict < or > (boundary excluded).

The system $\begin{cases}y \le 3 \\ y \ge -1\end{cases}$ has a feasible region that is:

The first inequality shades below $y=3$; the second shades above $y=-1$. Where do those regions overlap?
$y \le 3$ covers everything at or below $y=3$; $y \ge -1$ covers everything at or above $y=-1$. The overlap is the horizontal strip $-1 \le y \le 3$.

Find the corner point of the system $\begin{cases}y \le -x+4 \\ y \le x+2\end{cases}$.

Set the two right-hand sides equal: $-x+4 = x+2$, then solve for $x$.
Set $-x+4 = x+2$: $2 = 2x$, so $x=1$. Then $y = 1+2 = 3$. Corner point: $(1,3)$. Verify: $3 \le -1+4=3$ ✓ and $3 \le 1+2=3$ ✓.

The system $\begin{cases}y > 2x+1 \\ y < 2x-3\end{cases}$ has:

Both lines have slope 2 — they are parallel. Draw them and think about which side each inequality shades.
Both boundary lines have slope 2 (parallel). $y > 2x+1$ shades above the upper line; $y < 2x-3$ shades below the lower line. These regions never overlap, so there is no solution.

The system $\begin{cases}x \ge 0 \\ y \ge 0 \\ y \le -x+4\end{cases}$ forms a triangular feasible region. What are the corner points?

The three corner points come from intersecting pairs of boundary lines: the x-axis, y-axis, and the line $y = -x+4$.
Intersections: axes meet at $(0,0)$; $y=-x+4$ meets x-axis ($y=0$) at $x=4$ → $(4,0)$; meets y-axis ($x=0$) at $y=4$ → $(0,4)$. Triangle: $(0,0)$, $(4,0)$, $(0,4)$.

Is $(-1,2)$ in the feasible region of $\begin{cases}y < x+4 \\ y \ge 2x+3\end{cases}$?

Substitute $x=-1, y=2$ into each inequality and check.
$2 < -1+4 = 3$ ✓ and $2 \ge 2(-1)+3 = 1$ ✓. Both inequalities pass, so $(-1,2)$ is in the feasible region.

Find the $x$-coordinate of the corner point of $\begin{cases}y \le 2x+1 \\ y \ge -x+7\end{cases}$.

Set $2x+1 = -x+7$ and solve for $x$.
$2x+1 = -x+7 \Rightarrow 3x = 6 \Rightarrow x = 2$. The corner point has $x$-coordinate 2.

Using the corner point from the previous problem ($x=2$), find the $y$-coordinate.

Substitute $x=2$ into either boundary equation, e.g. $y = 2(2)+1$.
$y = 2(2)+1 = 5$. (Verify with the other: $-2+7 = 5$ ✓.) The corner point is $(2, 5)$.
