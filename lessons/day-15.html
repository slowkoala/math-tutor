Day 15 Functions

Introduction to Functions

Key Vocabulary

Function
A relation where each input (x) has exactly one output (y). Every x maps to exactly one y.
Domain
The set of all allowed input values (x-values) for a function.
Range
The set of all possible output values (y-values) produced by a function.
Function Notation
\(f(x)\) — read "f of x" — means the output of function f when the input is x.
Vertical Line Test
A graph represents a function if and only if every vertical line crosses the graph at most once.
Mapping Diagram
An arrow diagram showing how each element of the domain maps to an element of the range.

What Makes a Function?

A function assigns exactly one output to every input. Think of it as a machine: you put in a number, you always get the same result back.

Vertical Line Test: Draw (or imagine) vertical lines through a graph. If any vertical line hits the graph more than once, it is not a function.

Worked Examples

Given \(f(x) = 3x - 2\), find \(f(4)\) and \(f(-1)\).

\(f(4)\): Substitute \(x = 4\): \(3(4) - 2 = 12 - 2 = 10\).

\(f(-1)\): Substitute \(x = -1\): \(3(-1) - 2 = -3 - 2 = -5\).

Does the table represent a function?
x1223
y57911

Check for repeated x-values: The input \(x = 2\) appears twice, giving outputs 7 and 9.

Conclusion: This is not a function — one input (2) maps to two different outputs.

Find the domain and range of \(f(x) = x^2\) if the domain is \(\{-2, -1, 0, 1, 2\}\).

Domain (given): \(\{-2, -1, 0, 1, 2\}\)

Evaluate:
\(f(-2) = 4,\quad f(-1) = 1,\quad f(0) = 0,\quad f(1) = 1,\quad f(2) = 4\)

Range (unique outputs): \(\{0, 1, 4\}\)
