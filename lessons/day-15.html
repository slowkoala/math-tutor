Day 15 Functions

Introduction to Functions

Vocabulary

Function
A relation where each input (x) has exactly one output (y). Every x maps to exactly one y.
Domain
The set of all allowed input values (x-values) for a function.
Range
The set of all possible output values (y-values) produced by a function.
Function Notation
$f(x)$ — read "f of x" — means the output of function f when the input is x.
Vertical Line Test
A graph represents a function if and only if every vertical line crosses the graph at most once.
Mapping Diagram
An arrow diagram showing how each element of the domain maps to an element of the range.

Concept

A function assigns exactly one output to every input. Think of it as a machine: you put in a number, you always get the same result back.

A function (left): each input maps to exactly one output. Not a function (right): input 2 maps to two outputs.

💡
Vertical Line Test: Draw (or imagine) vertical lines through a graph. If any vertical line hits the graph more than once, it is not a function.

Worked Examples

Example 1 — Evaluate a function at given inputs

Given $f(x) = 3x - 2$, find $f(4)$ and $f(-1)$.

1
$f(4)$: Substitute $x = 4$: $3(4) - 2 = 12 - 2 = 10$.
2
$f(-1)$: Substitute $x = -1$: $3(-1) - 2 = -3 - 2 = -5$.
$f(4) = 10$ and $f(-1) = -5$.

Example 3 — Find domain and range from a restricted domain

Find the domain and range of $f(x) = x^2$ if the domain is $\{-2, -1, 0, 1, 2\}$.

1
Domain (given): $\{-2, -1, 0, 1, 2\}$
2
Evaluate:
$f(-2) = 4,\quad f(-1) = 1,\quad f(0) = 0,\quad f(1) = 1,\quad f(2) = 4$
3
Range (unique outputs): $\{0, 1, 4\}$
Domain: $\{-2,-1,0,1,2\}$. Range: $\{0,1,4\}$.

Practice Problems

If $f(x) = 2x + 5$, find $f(3)$.

Replace $x$ with 3 in the formula: $2(3) + 5$.
$f(3) = 2(3) + 5 = 6 + 5 = 11$.

If $f(x) = x^2 - 1$, find $f(-2)$.

$(-2)^2 = 4$. Now subtract 1.
$f(-2) = (-2)^2 - 1 = 4 - 1 = 3$.

Which set of ordered pairs represents a function?

Check each set for repeated x-values. A function cannot have the same x paired with different y values.
Set A: $x=1$ appears twice (outputs 2 and 3) — not a function. Set B: all x-values unique — function. Set C: $x=2$ appears twice — not a function. Set D: $x=0$ appears twice — not a function.

A circle graphed on the coordinate plane — does it pass the vertical line test?

Imagine drawing a vertical line through the center of a circle. How many points does it cross?
A vertical line through the center of a circle crosses it at two points (top and bottom). Since at least one vertical line hits more than one point, it fails the vertical line test — a circle is not a function.

For $f(x) = 5x - 3$, which best describes the domain?

Is there any real number you cannot substitute for $x$ in $5x - 3$?
$f(x) = 5x - 3$ is a linear function with no restrictions (no division by zero, no square roots of negatives). Any real number can be input, so the domain is all real numbers.

For $f(x) = 4 - x$ with domain $\{0,1,2,3,4\}$, which value is in the range?

Evaluate $f(x) = 4-x$ for each value in the domain $\{0,1,2,3,4\}$ and list the outputs.
Outputs: $f(0)=4, f(1)=3, f(2)=2, f(3)=1, f(4)=0$. Range = $\{0,1,2,3,4\}$. The only answer choice in this range is 2.

If $g(x) = 3x - 1$, find $g(-3)$.

Substitute $x = -3$: $3(-3) - 1$.
$g(-3) = 3(-3) - 1 = -9 - 1 = -10$.

The table shows $x$: 1, 2, 3, 4 and $y$: 2, 4, 6, 8. Is this a function?

Check: does any x-value repeat? Having different y-values is not a problem — having the same x with different y's would be.
Each x-value (1, 2, 3, 4) appears exactly once, each with a unique output. There are no repeated x-values, so this table represents a function.

If $f(x) = x^2 + x$, find $f(2)$.

$f(2) = 2^2 + 2$. Compute $2^2$ first.
$f(2) = 2^2 + 2 = 4 + 2 = 6$.

Which statement best describes a function?

Think about the definition: a function is a rule that assigns exactly one ___ to every ___.
By definition, a function assigns exactly one output to every input. Multiple inputs can share the same output (that's fine), but no single input can give two different outputs.
