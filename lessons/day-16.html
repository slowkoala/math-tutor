Day 16 Functions

Evaluating & Comparing Functions

Key Vocabulary

Rate of Change (Function)
How fast the output changes per unit of input: \(\dfrac{\Delta y}{\Delta x} = \dfrac{f(b)-f(a)}{b-a}\).
Initial Value
The output when the input is zero, \(f(0)\) — the y-intercept of the function.
Linear Function
A function whose graph is a straight line; has a constant rate of change (slope).
Increasing / Decreasing
A function is increasing if outputs rise as inputs rise; decreasing if outputs fall.
Comparing Functions
Functions can be compared by slope, y-intercept, or values at specific x — even when given in different forms.
Function Representations
A function can be described as an equation, a table, a graph, or a verbal description.

Comparing Functions in Different Forms

Two functions may be given in different forms. To compare them, extract the slope and y-intercept from each.

Function A (equation)
\(f(x) = 2x + 1\)
Slope: 2, y-int: 1
Function B (table)
x012
y369
Slope: 3, y-int: 3

Function B has a steeper slope (3 > 2) and a higher y-intercept (3 > 1).

Key comparison questions: Which has a greater rate of change? Which has a higher initial value? Which will eventually be greater?

Worked Examples

Find the average rate of change of \(f(x) = x^2 + 1\) from \(x = 1\) to \(x = 3\).

Step 1: \(f(1) = 1^2 + 1 = 2\)

Step 2: \(f(3) = 3^2 + 1 = 10\)

Step 3: \(\dfrac{f(3)-f(1)}{3-1} = \dfrac{10-2}{2} = \dfrac{8}{2} = 4\)

The average rate of change is 4.

Function A: \(f(x) = 4x - 3\). Function B is shown in the table:
x024
y1917
Which has a greater slope?

Function A slope: 4 (coefficient of x).

Function B slope: \(\dfrac{9-1}{2-0} = \dfrac{8}{2} = 4\). Same slope!

Compare y-intercepts: A: \(-3\); B: \(1\) (from table, when x=0, y=1).

The slopes are equal; Function B has a higher y-intercept.

Is \(f(x) = -2x + 5\) increasing or decreasing? Find \(f(0)\) and \(f(3)\).

Slope: \(-2\) (negative) → the function is decreasing.

\(f(0)\): \(-2(0)+5 = 5\)

\(f(3)\): \(-2(3)+5 = -6+5 = -1\)

As x increases from 0 to 3, y decreases from 5 to −1. Confirmed decreasing.
