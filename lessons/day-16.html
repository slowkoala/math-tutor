Day 16 Functions

Evaluating & Comparing Functions

Vocabulary

Rate of Change (Function)
How fast the output changes per unit of input: $\dfrac{\Delta y}{\Delta x} = \dfrac{f(b)-f(a)}{b-a}$.
Initial Value
The output when the input is zero, $f(0)$ — the y-intercept of the function.
Linear Function
A function whose graph is a straight line; has a constant rate of change (slope).
Increasing / Decreasing
A function is increasing if outputs rise as inputs rise; decreasing if outputs fall.
Comparing Functions
Functions can be compared by slope, y-intercept, or values at specific x — even when given in different forms.
Function Representations
A function can be described as an equation, a table, a graph, or a verbal description.

Concept

Two functions may be given in different forms. To compare them, extract the slope and y-intercept from each.

Function A (equation)
$f(x) = 2x + 1$
Slope: 2, y-int: 1
Function B (table)
x012
y369
Slope: 3, y-int: 3

Function B has a steeper slope (3 > 2) and a higher y-intercept (3 > 1).

Function B (green) is steeper and starts higher than Function A (blue).

💡
Key comparison questions: Which has a greater rate of change? Which has a higher initial value? Which will eventually be greater?

Worked Examples

Example 1 — Average rate of change of a quadratic

Find the average rate of change of $f(x) = x^2 + 1$ from $x = 1$ to $x = 3$.

1
$f(1) = 1^2 + 1 = 2$
2
$f(3) = 3^2 + 1 = 10$
3
$\dfrac{f(3)-f(1)}{3-1} = \dfrac{10-2}{2} = \dfrac{8}{2} = 4$
The average rate of change is 4.
Example 2 — Compare slopes of two functions in different forms

Function A: $f(x) = 4x - 3$. Function B is shown in the table:

x024
y1917

Which has a greater slope?

1
Function A slope: 4 (coefficient of x).
2
Function B slope: $\dfrac{9-1}{2-0} = \dfrac{8}{2} = 4$. Same slope!
3
Compare y-intercepts: A: $-3$; B: $1$ (from table, when x=0, y=1).
The slopes are equal (both 4); Function B has a higher y-intercept.
Example 3 — Determine if a function is increasing or decreasing

Is $f(x) = -2x + 5$ increasing or decreasing? Find $f(0)$ and $f(3)$.

1
Slope: $-2$ (negative) → the function is decreasing.
2
$f(0)$: $-2(0)+5 = 5$
3
$f(3)$: $-2(3)+5 = -6+5 = -1$
Decreasing. As x increases from 0 to 3, y decreases from 5 to −1.

Practice Problems

If $f(x) = 5x - 2$, find $f(4)$.

Substitute $x = 4$: $5(4) - 2$.
$f(4) = 5(4) - 2 = 20 - 2 = 18$.

Find the average rate of change of $f(x) = 2x + 3$ from $x = 1$ to $x = 5$.

For a linear function, the average rate of change equals the slope (coefficient of x).
$f(1) = 5$, $f(5) = 13$. Rate $= \frac{13-5}{5-1} = \frac{8}{4} = 2$. (This equals the slope of the linear function.)

Is $f(x) = -3x + 6$ increasing or decreasing?

The sign of the slope (coefficient of x) determines whether a linear function increases or decreases.
The slope is $-3$, which is negative. A negative slope means the function decreases as x increases.

Function A: $f(x) = 3x + 1$. Function B has slope 5. Which function is steeper?

Compare the slopes directly: Function A has slope 3, Function B has slope 5.
Function A slope = 3; Function B slope = 5. Since 5 > 3, Function B is steeper.

Find the average rate of change of $f(x) = x^2$ from $x = 2$ to $x = 4$.

Compute $f(4) - f(2)$ divided by $4 - 2$.
$f(2) = 4$, $f(4) = 16$. Rate $= \frac{16-4}{4-2} = \frac{12}{2} = 6$.

A table shows $x$: 0, 1, 2, 3 and $y$: 4, 7, 10, 13. Which equation represents this function?

The y-intercept is the y-value when $x=0$. The slope is the change in y per unit change in x.
When $x=0$, $y=4$ → y-intercept = 4. Slope $= \frac{7-4}{1-0} = 3$. Equation: $f(x) = 3x + 4$. Verify: $f(2)=10$ ✓, $f(3)=13$ ✓.

Compare $f(x) = 2x + 3$ and $g(x) = 2x - 1$. Which statement is true?

Both functions have the same slope. Their graphs are parallel lines. The one with the higher y-intercept is always above the other.
Both have slope 2 (parallel lines). $f$ has y-intercept 3; $g$ has y-intercept $-1$. So $f(x) = g(x) + 4$ — f is always 4 units above g. They never intersect.

What is the initial value (y-intercept) of $f(x) = 7x - 4$?

The initial value is $f(0)$. Substitute $x = 0$.
$f(0) = 7(0) - 4 = -4$. The y-intercept (initial value) is $-4$.

Function A: slope 2, y-intercept 5. Function B: slope 4, y-intercept 0. At $x = 3$, which has the greater value?

Evaluate each function at $x=3$: A gives $2(3)+5$ and B gives $4(3)+0$.
$f(3) = 2(3)+5 = 11$; $g(3) = 4(3)+0 = 12$. Function B is greater at $x=3$.

Find the average rate of change of $f(x) = 3x^2 - x$ from $x = 0$ to $x = 2$.

Compute $f(0)$ and $f(2)$, then divide the difference by $2 - 0 = 2$.
$f(0) = 0$; $f(2) = 3(4) - 2 = 10$. Rate $= \frac{10-0}{2-0} = \frac{10}{2} = 5$.
