Day 18 Functions

Creating Functions from Contexts

Key Vocabulary

Modeling
Writing a mathematical function that represents a real-world situation.
Constraint
A restriction on the domain based on the real-world context (e.g., time cannot be negative).
Reasonable Domain
The set of input values that make sense for the given real-world situation.
Fixed Cost
A cost that does not change with the number of units — it is the y-intercept in a linear model.
Variable Cost
A cost that changes based on the number of units — it is the slope in a linear model.
Piecewise Function
A function defined by different rules on different parts of its domain.

From Words to Functions

Real-world problems often describe a relationship in words. Your job is to identify the input, output, rate, and starting value, then write the function.

Choosing the model type:
  • Linear: constant rate of change ("per," "each," "every")
  • Exponential: percent growth/decay, doubling/halving
  • Piecewise: different rules for different intervals

Worked Examples

A gym charges a $30 membership fee plus $10 per visit. Write a function for total cost \(C\) after \(v\) visits, then find the cost for 8 visits.

Input: \(v\) = number of visits.

Output: \(C\) = total cost in dollars.

Rate: $10 per visit (slope = 10).

Start: $30 membership (y-intercept = 30).

Model: \(C(v) = 10v + 30\)

8 visits: \(C(8) = 10(8)+30 = 80+30 = \$110\)

A savings account starts with $500 and earns 4% interest annually. Write a model and find the balance after 3 years (round to nearest dollar).

Model type: Exponential growth (percent rate).

Growth factor: \(b = 1 + 0.04 = 1.04\).

Model: \(B(t) = 500 \cdot (1.04)^t\)

After 3 years: \(B(3) = 500 \cdot (1.04)^3 \approx 500 \cdot 1.1249 \approx \$562\)

A taxi charges $2.50 flat fee plus $1.80 per mile for the first 5 miles, then $1.20 per mile after that. Write a piecewise function for cost \(C(m)\).

For \(0 \le m \le 5\): \(C(m) = 2.50 + 1.80m\)

For \(m > 5\): First 5 miles cost \(2.50 + 9.00 = 11.50\). Each additional mile costs $1.20.

\(C(m) = 11.50 + 1.20(m - 5)\) for \(m > 5\).

Piecewise: \(C(m) = \begin{cases} 2.50 + 1.80m & 0 \le m \le 5 \\ 11.50 + 1.20(m-5) & m > 5 \end{cases}\)

Practice Problems

A phone plan costs $25 per month plus $0.05 per text message. What is the total cost (in dollars) for 60 text messages?
A candle is 100 cm tall and burns 5 cm per hour. Which function gives its height \(h\) after \(t\) hours?
  1. \(h(t) = 5t + 100\)
  2. \(h(t) = 100t - 5\)
  3. \(h(t) = 100 - 5t\)
  4. \(h(t) = 5(100 - t)\)
For the candle in the previous problem, what is the reasonable domain?
  1. \(t \ge 0\) (all non-negative hours)
  2. \(0 \le t \le 20\) (stops when candle is gone)
  3. \(-20 \le t \le 20\)
  4. \(0 \le t \le 100\)
$1,000 invested at 6% annual growth. Which is the correct model?
  1. \(V(t) = 1000 + 6t\)
  2. \(V(t) = 1000 \cdot (0.06)^t\)
  3. \(V(t) = 1000 \cdot (0.94)^t\)
  4. \(V(t) = 1000 \cdot (1.06)^t\)
A store pays $40 rent and sells items for $8 each. How many items must be sold to break even (profit = 0)?
A table shows: \(x = 0, 1, 2\) and \(y = 7, 10, 13\). Using the pattern, find \(f(5)\).
A delivery service charges: \(C(m) = \begin{cases}3 + 2m & m \le 4 \\ 11 + 1.5(m-4) & m > 4\end{cases}\). Find \(C(6)\).
An employee earns $12 per hour plus a $50 weekly bonus. Which function gives weekly earnings for \(h\) hours?
  1. \(E(h) = 50h + 12\)
  2. \(E(h) = 12h + 50\)
  3. \(E(h) = 12 \cdot 50^h\)
  4. \(E(h) = 50 \cdot (1.12)^h\)
"A town's population triples every decade." Which type of model fits?
  1. Linear — constant rate
  2. Exponential — multiplicative rate
  3. Piecewise — different rules
  4. Cannot be modeled mathematically
A car rental costs $40 per day plus $0.20 per mile. What is the total cost (in dollars) for 3 days and 150 miles?
