Day 21 Polynomials & Quadratics

Graphing Quadratic Functions

Key Vocabulary

Parabola
The U-shaped graph of a quadratic function \(f(x) = ax^2 + bx + c\).
Vertex
The turning point of the parabola — the minimum if \(a>0\), maximum if \(a<0\).
Axis of Symmetry
The vertical line through the vertex: \(x = -\dfrac{b}{2a}\).
Vertex Form
\(f(x) = a(x-h)^2 + k\) — vertex is at \((h, k)\).
Opens Up / Down
If \(a > 0\), parabola opens up (min vertex). If \(a < 0\), opens down (max vertex).
x-intercepts
Points where the parabola crosses the x-axis; the zeros/roots of the quadratic.

Key Features of a Parabola

For \(f(x) = ax^2 + bx + c\):

  • Vertex x-coordinate: \(x = -\dfrac{b}{2a}\)
  • Vertex y-coordinate: substitute x back into f(x)
  • y-intercept: \(f(0) = c\)
  • x-intercepts: factor or use quadratic formula

Worked Examples

Find the vertex and axis of symmetry of \(f(x) = x^2 - 6x + 8\).

Axis of symmetry: \(x = -\dfrac{b}{2a} = -\dfrac{-6}{2(1)} = 3\)

Vertex y: \(f(3) = 9 - 18 + 8 = -1\)

Vertex: \((3, -1)\). Axis of symmetry: \(x = 3\).

Write \(f(x) = x^2 - 4x + 7\) in vertex form.

Complete the square:

\(f(x) = (x^2 - 4x \phantom{+4}) + 7\)

Add and subtract \(\left(\tfrac{4}{2}\right)^2 = 4\):

\(f(x) = (x^2 - 4x + 4) + 7 - 4 = (x-2)^2 + 3\)

Vertex form: \((x-2)^2 + 3\). Vertex: \((2, 3)\).

A ball is thrown upward. Its height in feet is \(h(t) = -16t^2 + 64t + 5\). What is the maximum height?

The maximum occurs at the vertex (since \(a = -16 < 0\)).

t at vertex: \(t = -\dfrac{64}{2(-16)} = -\dfrac{64}{-32} = 2\) seconds.

Max height: \(h(2) = -16(4) + 64(2) + 5 = -64 + 128 + 5 = 69\) feet.
