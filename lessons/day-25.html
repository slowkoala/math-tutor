Day 25 Geometry

Pythagorean Theorem

Vocabulary

Pythagorean Theorem
In a right triangle: $a^2 + b^2 = c^2$, where $c$ is the hypotenuse (longest side).
Hypotenuse
The side opposite the right angle — always the longest side of a right triangle.
Leg
Either of the two shorter sides of a right triangle that form the right angle.
Pythagorean Triple
A set of three whole numbers satisfying $a^2 + b^2 = c^2$. Common: 3-4-5, 5-12-13, 8-15-17.
Converse of Pythagorean Theorem
If $a^2 + b^2 = c^2$, then the triangle is a right triangle.
Distance Formula
$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$ — derived from the Pythagorean Theorem.

Concept

A 3-4-5 right triangle: $3^2 + 4^2 = 9 + 16 = 25 = 5^2$.

💡
Common Pythagorean Triples (memorize these!):
3-4-5  |  5-12-13  |  8-15-17  |  7-24-25
Any multiple also works: 6-8-10, 9-12-15, …

Worked Examples

Example 1 — Find the hypotenuse

Find the hypotenuse of a right triangle with legs 5 and 12.

1
Write the theorem: $a^2 + b^2 = c^2$
2
Substitute: $5^2 + 12^2 = c^2 \Rightarrow 25 + 144 = 169$
3
Take the square root: $c = \sqrt{169} = 13$
Hypotenuse $= 13$  (this is a 5-12-13 Pythagorean triple)
Example 2 — Find a missing leg

A right triangle has hypotenuse 17 and one leg 8. Find the other leg.

1
Set up: $a^2 + 8^2 = 17^2$
2
$a^2 + 64 = 289$
3
$a^2 = 225 \Rightarrow a = 15$
Other leg $= 15$  (8-15-17 triple)
Example 3 — Distance between two points

Find the distance between $(-3,1)$ and $(5,7)$.

1
Distance formula: $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
2
Substitute: $d = \sqrt{(5-(-3))^2 + (7-1)^2} = \sqrt{8^2 + 6^2}$
3
$= \sqrt{64 + 36} = \sqrt{100} = 10$
Distance $= 10$

Practice Problems

Find the hypotenuse of a right triangle with legs 6 and 8.

$6^2 + 8^2 = 36 + 64 = ?$ Then take the square root.
$\sqrt{36+64} = \sqrt{100} = 10$  (6-8-10 is a multiple of 3-4-5).

A right triangle has hypotenuse 13 and one leg 5. Find the other leg.

$a^2 + 5^2 = 13^2 \Rightarrow a^2 = 169 - 25$
$a^2 = 169 - 25 = 144 \Rightarrow a = 12$  (5-12-13 triple).

Is 7, 24, 25 a Pythagorean triple?

Check: does $7^2 + 24^2 = 25^2$? Compute each side.
$49 + 576 = 625$ and $25^2 = 625$. ✓ It is a Pythagorean triple.

A triangle has sides 9, 12, and 15. Is it a right triangle?

Use the converse: check whether $a^2 + b^2 = c^2$ with 15 as the longest side.
$81 + 144 = 225 = 15^2$ ✓. This is a 3-4-5 triple scaled by 3.

Find the distance between $(0,0)$ and $(5,12)$.

$d = \sqrt{5^2 + 12^2}$. Do you recognize this triple?
$\sqrt{25 + 144} = \sqrt{169} = 13$  (5-12-13 triple).

Find the hypotenuse of a right triangle with both legs equal to 1. Round to the nearest hundredth.

$c = \sqrt{1^2 + 1^2} = \sqrt{2}$. What is $\sqrt{2}$ rounded to the nearest hundredth?
$c = \sqrt{2} \approx 1.41$. This is an irrational number.

Find the distance between $(1,2)$ and $(4,6)$.

$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{3^2 + 4^2}$
$\sqrt{9 + 16} = \sqrt{25} = 5$  (3-4-5 triple).

A 10-foot ladder leans against a wall. The base is 6 feet from the wall. How high up the wall does the ladder reach?

The ladder is the hypotenuse (10 ft), the base is one leg (6 ft). Find the missing leg.
$h^2 + 6^2 = 10^2 \Rightarrow h^2 = 100 - 36 = 64 \Rightarrow h = 8$ ft.

A triangle has sides 5, 7, and 9. Is it a right triangle?

Check: $5^2 + 7^2 = 25 + 49 = ?$ Compare to $9^2 = 81$.
$25 + 49 = 74 \ne 81$. Since $a^2 + b^2 \ne c^2$, this is not a right triangle.

Find the hypotenuse of a right triangle with legs 20 and 21.

$c = \sqrt{20^2 + 21^2} = \sqrt{400 + 441}$
$\sqrt{841} = 29$  (20-21-29 is a Pythagorean triple).
